## 프로젝트 개요
#### 개요
  - 오픈 소스 API를 활용하여 사용한 상황에 맞는 답변 제시, 어투 제시 기능 개발
#### 목적
  - 사용자가 다양한 언어와 어투로 공식적이고 정중한 이메일을 쉽게 작성할 수 있도록 지원
#### 타겟 사용자
  - 이메일 작성의 효율성을 높이고 싶은 직장인 및 다국어 이메일 작성이 필요한 사용자
#### 핵심 기능
1. 이메일 답변 생성 기능
2. 다양한 브라우저 환경에서 제공
